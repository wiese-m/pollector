## Introduction

Pollector - UDP server with ClickHouse writer for HFT market data. Trades are serialized as simple strings.

## Quick Start

1. Run UDP server & initialize CH database:

```shell
docker compose up --build
```

2. Run any client from `examples` directory or write your own:

```shell
uv run -m examples.binance_perp
```

3. Trades will be saved in `market_data.trades` table:

```clickhouse
select * from market_data.trades order by ts_recv desc limit 20;
```
